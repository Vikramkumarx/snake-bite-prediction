üìà Model Training Process

  1. Data Collection: Gathered ~2,000 images per species using web scraping
  2. Data Preprocessing: Resizing, normalization, and augmentation
  3. Transfer Learning: Fine-tuned ResNet34 pre-trained on ImageNet
  4. Training: 5 epochs with progressive learning rate reduction
  5. Validation: Achieved 85%+ accuracy on validation set
